Calculating CAT4016 Current¶

The current for the different CAT4016 should be dimensioned in a way that guarantees that every chip sinks a current that is “significantly” bigger than the current sunk by the next smaller chip. Unfortunately it does not make sense to add the individual currents because our eye perceives luminosity logarithmically (and the intensity is linearly depend on the current).

The following plot shows some possible values where each CAT4016 can be fine-tuned with a potentiometer to manually set a precise current that fulfills the above-mentioned criteria.
